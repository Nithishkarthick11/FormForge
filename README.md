FormForge is a fitness web app that helps users learn the correct exercise form with images and step-by-step instructions.
Features:
  - Search and filter exercises
  - Save favorites (Firebase)
  - Dark / Light mode toggle
  - Favorite count badge
  - Responsive UI
Tech Stack:
  - React (Vite)
  - Firebase Auth & Firestore
  - React Router
  - CSS
